# 🚀 1. AJOUTE ÇA DANS TON HTML (OBLIGATOIRE)

Dans **tous tes fichiers HTML** (`dashboard.html`, `login.html`, etc.), ajoute dans `<head>` :

```html
<meta name="viewport" content="width=device-width, initial-scale=1.0">
```

👉 Sans ça, ton site ne sera JAMAIS responsive

---

# 🎯 2. UTILISE FLEXBOX (très important)

Dans ton CSS, évite les positions fixes.

👉 Exemple :

❌ Mauvais :

```css
.container {
  width: 1200px;
}
```

✅ Bon :

```css
.container {
  width: 100%;
  max-width: 1200px;
  margin: auto;
}
```

---

# 📱 3. AJOUTE DES MEDIA QUERIES (clé du responsive)

👉 Exemple simple :

```css
@media (max-width: 768px) {
  body {
    padding: 10px;
  }

  .container {
    flex-direction: column;
  }

  .card {
    width: 100%;
  }
}
```

---

# 🔥 4. REND TES ÉLÉMENTS ADAPTATIFS

### Images

```css
img {
  max-width: 100%;
  height: auto;
}
```

### Boutons / Inputs

```css
input, button {
  width: 100%;
}
```

---

# 💡 5. UTILISE FLEX OU GRID

👉 Exemple FLEX :

```css
.row {
  display: flex;
  gap: 10px;
}

@media (max-width: 768px) {
  .row {
    flex-direction: column;
  }
}
```

---

# ⚡ 6. SOLUTION RAPIDE (je te recommande)

Si tu veux aller vite :

👉 utilise **Bootstrap**

Ajoute dans ton HTML :

```html
<link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.0/dist/css/bootstrap.min.css" rel="stylesheet">
```

Puis :

```html
<div class="container">
  <div class="row">
    <div class="col-md-6 col-12">Bloc 1</div>
    <div class="col-md-6 col-12">Bloc 2</div>
  </div>
</div>
```

👉 Sur PC : 2 colonnes
👉 Sur téléphone : 1 colonne
